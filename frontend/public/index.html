EMAS Formular - JSON Generator

Willkommen zum EMAS Formular-JSON-Generator! Hier können Sie Formularelemente per Drag-and-Drop hinzufügen, ihre Eigenschaften bearbeiten und das Design als JSON exportieren. Folgen Sie einfach den untenstehenden Schritten:

  1. Wählen Sie ein Element aus der Liste "Verfügbare Elemente" aus.
  2. Ziehen Sie es in den Bereich "Formular".
  3. Klicken Sie auf ein Element im "Formular", um seine Eigenschaften zu bearbeiten.
  4. Wenn Sie fertig sind, klicken Sie auf "Exportieren als JSON", um Ihr Design zu speichern.
Sie können ihr erstelltes Design in der Datenbank speichern und mit der gennerierten ID zu einem späteren Zeitpunkt weiter machen.
